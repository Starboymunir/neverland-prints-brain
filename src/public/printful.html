Fulfillment Flow

How Orders Are Fulfilled
1. Customer places order on Shopify
2. Shopify sends webhook to our backend
3. Backend looks up artwork image (Google Drive)
4. Backend creates Printful order via API
5. Printful prints, packs, ships to customer
No Printful product sync needed — each order is created on-demand with the artwork image. This works for unlimited products.
